# Frontend Practice Project

A simple frontend practice project created for learning and experimenting with basic HTML and CSS concepts.

## Overview

This project serves as a landing page that links to other practice applications and exercises. It is intended for educational purposes and frontend development practice only.

## Features

- Clean and minimal HTML structure
- External CSS styling support
- Navigation link to a practice application
- Mobile viewport configuration
- Reference to the original design/template source

## Project Structure

```text
project/
│
├── index.html
├── style.css
└── firstApp/
    └── app1.html
└── secondApp/
    └── app2.html
```

## Getting Started

1. Clone or download the project.
2. Open `index.html` in your preferred web browser.
3. Click the provided link to access the first practice page.
